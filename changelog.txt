27.07.18 v1.1.0:
	* При создании префикса вместо паузы в виде sleep используется
		вызов wineserver -w, что позволило исправить редкие ошибки
		связанные с запущенным wineserver при создании префикса
	* Удален код для скачивания winetricks
	* Добавлены параметры запуска:
		--syswine - для использования системного Wine
		--window - для включения виртуального рабочего стола
		--debug - для включения WINEDEBUG
	* Добавлены новые переменные
	* При наличии файла game_info/win64 скрипт автоматически создаст
		64-битный префикс
	* Добавлена функция включения/отключения DXVK
	* Добавлена функция восстановления разрешения экрана
	* Добавлена функция изменения аудиодрайвера на PulseAudio
	* Добавлена функция изменения версии windows
	* При создании префикса теперь автоматически выполняются файлы
		из каталога game_info/exe
	* Изменен вывод информации при запуске игры
	* При создании префкса создается каталог documents, в котором будут
		храниться все сохранения и настройки игр. Таким образом, при
		удалении префикса, настройки и сохранения игр не удаляются.
	* При изменении пользователя или Wine префикс автоматически
		пересоздается.
	* При наличии файла docopy в каталогах game_info/dir_$N, скрипт будет
		копировать файлы вместо создания символических ссылок
	* При наличии пустых файлов название.dll_nooverride или
		название.dll_register в каталоге game_info/dlls/special
		определенные файлы могут быть зарегистрированы скриптом с помощью
		regsvr32 или помещены в system32 без их переопределения
	* Каталог game_info/dlls теперь используется не только для dll файлов
		но и для всех других файлов, которые нужно поместить в system32
	* Настройки скрипта вынесены в отдельный файл (settings_start)
	* Копии скрипта start.sh с другими названиями будут использовать
		файлы game_info_НАЗВАНИЕСКРИПТА.txt и settings_НАЗВАНИЕСКРИПТА
		Например: game_info_start-launcher.txt и settings_start-launcher
	* Исправлено отключение CSMT
	* Добавлен скрипт для создания иконки на рабочем столе и в меню
		приложений
	* Другие мелкие исправления
	* Обновлена документация

18.09.18 v1.1.4:
	* Reg файлы теперь снова применяются до обработки dll файлов.
		Так как иначе могли возникать проблемы с регистрацией
		некоторых dll файлов.
	* Каждый reg файл теперь применяется отдельно. Это помогло
		избежать проблем с reg файлами, имеющими окончания строк,
		отличные от LF (Unix)
	* Дополнена коррекция переменной WINEARCH
	* Вызов скриптов вроде regedit, winecfg, winefile и т.д. заменен
		на вызов бинарника wine с соответствующим параметром
	* Добавлен вызов бинарника wine64 при регистрации dll и применении
		reg файлов. Что исправляет проблемы с dll и reg файлами
		на 64-битном префиксе (с syswow64).
